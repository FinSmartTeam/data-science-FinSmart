# 📊 Student Expense Dashboard

Dashboard ini dibuat untuk menganalisis pola pemasukan dan pengeluaran mahasiswa menggunakan Streamlit.

## ⚙️ Cara Menjalankan

1. Masuk ke folder project:
   cd Dashboard Capstone

2. Buat virtual environment:
   py -m venv venv

3. Aktifkan virtual environment:

   Windows:
   venv\Scripts\activate

   Mac/Linux:
   source venv/bin/activate

4. Install dependencies:
   pip install -r requirements.txt

5. Jalankan dashboard:
   streamlit run dashboard_capstone.py

## 📊 Dataset
Dataset yang digunakan adalah Student Expense Dataset yang berisi data pemasukan dan pengeluaran mahasiswa selama periode enam bulan.
